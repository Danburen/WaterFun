WaterFun — Interactive Community Forum Platform

WaterFun is a feature-rich community forum platform built with a gateway-centric microservice architecture, dedicated to providing a secure, smooth, and diverse interactive experience.

==== Platform Highlights ====

• Secure Authentication
  Password / SMS / Email login & register, dual-token (Access + Refresh),
  RS256 asymmetric JWT, device fingerprint binding.

• Account Security
  Set/reset password, bind/modify email & phone, captcha verification.

• User Profiles
  Profile editing, avatar upload (COS presigned URL + callback),
  follower system, privacy settings.

• Content & Interaction
  Post CRUD, category/tag filtering, comments, likes, bookmarks.

• Notification System
  System inbox, unread count, batch read, SSE real-time push,
  RabbitMQ async delivery.

• Ticket System
  Content report, appeal, feedback — unified endpoint, MQ decoupled.

• Moderation
  Post/image/text audit workflow, batch operations, audit history.

• Admin Dashboard
  User/role/permission management, site statistics, JVM monitoring,
  online users, ECharts visualization.

==== Tech Stack ====

  Framework: Spring Boot 4.0.x + Spring Cloud Gateway 2025.1.0
  Language: Java 22 + TypeScript 5.9
  Frontend: Nuxt 4 + Vue 3 (Client), Vue 3 + Vite 7 (Admin)
  UI: Element Plus
  Auth: Spring Security + OAuth2 Resource Server + RS256 JWT
  ORM: JPA / Hibernate + MySQL
  Cache: Caffeine (L1) → Redis (L2) → MySQL (L3)
  Queue: RabbitMQ
  Storage: Tencent COS (presigned URLs)
  SMS: Aliyun SMS
  Email: Resend + Spring Mail
