WaterFun — 多元化互动论坛社区

WaterFun 是一个功能丰富的社区论坛平台，采用网关为中心的微服务架构，致力于为用户提供安全、流畅、多元化的互动体验。

==== 平台特色 ====

• 安全可靠的身份认证
  支持密码/短信/邮箱登录注册，双令牌机制（Access Token + Refresh Token），
  RS256 非对称签名 JWT，设备指纹绑定，有效防止令牌盗用。

• 完善的账号安全体系
  密码设置/重置，邮箱/手机号绑定与修改，图形验证码防护。

• 丰富的用户系统
  个性化资料编辑，头像上传（COS 预签名 URL + 回调确认），
  关注/粉丝系统，隐私权限设置。

• 内容创作与互动
  帖子发布与管理，分类/标签筛选，评论互动，点赞收藏，
  打造优质的内容社区。

• 智能通知系统
  系统收件箱，未读计数，批量已读，SSE 实时推送，
  RabbitMQ 异步投递，消息不丢失。

• 工单与举报系统
  内容举报、账号申诉、意见建议 — 统一入口，
  MQ 异步解耦，管理员审核后自动执行处罚并通知。

• 内容审核机制
  帖子/图片/文本审核流程，批量操作，审核历史记录，
  RabbitMQ 回调策略。

• 封禁管理系统
  6 种处罚类型，AOP @BanCheck 注解自动拦截，
  自动延期、永久覆盖、解封管理。

• 管理后台
  用户/角色/权限管理，站点统计，JVM 监控，
  在线用户，ECharts 数据可视化。

==== 技术架构 ====

  框架：Spring Boot 4.0.x + Spring Cloud Gateway 2025.1.0
  语言：Java 22 + TypeScript 5.9
  前端：Nuxt 4 + Vue 3（用户端），Vue 3 + Vite 7（管理端）
  UI 库：Element Plus
  认证：Spring Security + OAuth2 Resource Server + RS256 JWT
  持久化：JPA / Hibernate + MySQL
  缓存：Caffeine（L1）→ Redis（L2）→ MySQL（L3）
  队列：RabbitMQ
  存储：腾讯云 COS（预签名 URL）
  短信：阿里云 SMS
  邮箱：Resend + Spring Mail

==== 项目结构 ====

  waterfun-gateway          API 网关（Spring Cloud Gateway）
  waterfun-service          用户端业务 API
  waterfun-admin-service    管理端 API
  waterfun-service-core     共享业务逻辑
  waterfun-common-lib       公共工具库
  waterfun-admin            管理后台（Vue 3 + Vite）
  waterfun-web-client       用户论坛（Nuxt 4 + Vue 3）
  waterfun-web-core         前端共享库
