# 关于 WaterFun

WaterFun 是一个多元化互动论坛社区，采用网关为中心的微服务架构，致力于为用户提供安全、流畅、丰富的互动体验。无论您是寻找技术交流、兴趣分享还是日常闲聊，WaterFun 都为您提供了一个开放、包容的交流空间。

## 平台特色

### 安全可靠的身份认证

WaterFun 采用双令牌认证机制，结合 RS256 非对称签名 JWT 和 Redis JTI 实时校验，确保每一次请求都经过严格的身份验证。设备指纹绑定技术有效防止令牌盗用，为您的账号安全保驾护航。

### 完善的账号安全体系

我们提供完整的账号安全管理功能，包括密码设置与重置、邮箱和手机号的绑定与修改，以及图形验证码防护。每一步操作都有安全保障，让您用得放心。

### 丰富的用户系统

每位用户都可以打造个性化的个人主页，支持自定义头像、昵称和个人简介。关注与粉丝系统让您轻松追踪感兴趣的用户，隐私权限设置则让您完全掌控个人信息的可见范围。

### 内容创作与互动

在 WaterFun，您可以自由发布和分享内容。平台支持帖子发布与管理、分类和标签筛选、评论互动、点赞和收藏等功能，为您打造一个优质的内容创作和交流空间。

### 智能通知系统

系统收件箱集成了未读计数、批量已读和 SSE 实时推送功能，结合 RabbitMQ 异步投递，确保每一条消息都能及时送达，不遗漏任何重要通知。

### 工单与举报系统

内容举报、账号申诉、意见建议通过统一入口提交，由 MQ 异步解耦处理。管理员审核后自动执行相应操作并通知您结果，流程高效透明。

### 社区管理

WaterFun 配备了完善的内容审核机制，支持帖子、图片、文本等多类型审核流程。封禁管理系统提供 6 种处罚类型，通过 AOP @BanCheck 注解自动拦截违规行为，同时支持自动延期、永久覆盖和灵活的账号解封管理。

### 管理后台

面向管理员的综合控制面板，涵盖用户、角色、权限管理，站点统计分析，JVM 实时监控，在线用户查看，以及 ECharts 数据可视化图表，让社区运营更加轻松高效。

## 技术架构

**框架**：Spring Boot 4.0.x + Spring Cloud Gateway 2025.1.0

**语言**：Java 22 + TypeScript 5.9

**前端**：Nuxt 4 + Vue 3（用户端），Vue 3 + Vite 7（管理端）

**UI 库**：Element Plus

**认证**：Spring Security + OAuth2 Resource Server + RS256 JWT

**持久化**：JPA / Hibernate + MySQL

**缓存**：Caffeine（L1）→ Redis（L2）→ MySQL（L3）三级缓存

**队列**：RabbitMQ 异步消息

**存储**：腾讯云 COS 预签名 URL

**短信**：阿里云 SMS

**邮箱**：Resend + Spring Mail
