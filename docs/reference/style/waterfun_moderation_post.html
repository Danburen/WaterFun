Vue3 性能优化实战指南

最近在项目中遇到了一些 Vue3 的性能瓶颈，分享一下我的优化经验。

1. 使用 shallowRef 替代 ref

对于不需要深度响应式的大型对象，使用 shallowRef 可以显著提升性能。

const state = shallowRef({ count: 0 });
// 只触发顶层响应式，不递归监听

2. 合理使用 computed

避免在模板中直接调用复杂计算，使用 computed 缓存结果。

性能优化是一个持续的过程，需要根据实际场景选择合适的方案。
